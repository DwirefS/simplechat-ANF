Key Capabilities

Multi-Protocol Access

Access the same data via NFS, SMB, and Object REST API (S3-compatible) simultaneously.

Zero Data Movement

Enterprise NAS data directly accessible to AI applications. No ETL pipelines needed.

Powered by ONTAP

Instant snapshots, cross-region replication, storage tiering, sub-millisecond latency.

Azure AI Integration

Native integration with Azure AI Search, Databricks, OneLake, and AI Foundry.

Enterprise Value Proposition

Single Source of Truth — One copy of data serves file sharing, data science, and AI/RAG workloads.
Accelerate Time-to-Value — Drop a file on the SMB share, query it in the AI chatbot seconds later.
Simplify Architecture — Eliminate complexity of syncing between file storage and object storage.
Enterprise Compliance — SAP HANA, GDPR, HIPAA, SOC certified.

What is SimpleChat?

SimpleChat is a Microsoft-created demo application that showcases enterprise AI capabilities using Azure services. This fork extends SimpleChat with Azure NetApp Files as the storage layer.

RAG-Powered Chat

Upload documents and chat with AI that grounds responses in your data using retrieval-augmented generation.

Document Management

Personal and group workspaces for organizing documents with full version control.

Enterprise Security

Azure Entra ID authentication, RBAC, content safety, and compliance features.

Multi-Format Support

PDF, Word, Excel, PowerPoint, images, audio, and video file processing.
